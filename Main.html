WELCOME TO SOFTWARE ENGINEERING

Software engineering as the name implies is about the engineering aspects of designing, developing and maintaining a software. It was introduced as a system to avoid development issues when building a software. In Software engineering, a software is developed correctly and consistently to meet its requirements within specified timeline and budget.

Just like in every other profession, let us take construction for example. If we were to build a house, we would need to plan how it would be built and we would start with designing the house structure and pattern on a piece of paper to provide guidelines to all personnel on the project. We would also need to assemble a team of architects, carpenters, builders who would also bring in their own tools and equipment such as tractors, and materials like sand, water, stones, wood etc, depending on the house specification. To make the house development successful, the whole team would need to work together for a certain period of time and every hand will be on deck to get the house built. After the house is built, a team of quality assurance will visit the house to check if the house is built according to approved specifications.

Software engineering is similar to the example in the paragraph above. To build a software project, it also requires planning, designing, developing, testing and maintaining the software within a specified project timeline.

VIRTUAL CLASS CSS

Now that we have learnt HTML and created a web page, let us take a look at how we can make our web pages prettier using CSS otherwise our web pages will look boring without it.

CSS means Cascading Style Sheet and it is used for making HTML documents look finer, because, by default most browsers will display your web pages with black text using Times New Roman font, and the other page contents such as lists, links, images, etc will appear in their default state.

We use CSS to make web pages look better and more appealing to our site visitors. To add CSS to a web page, you can add it directly to an HTML tag using the style attribute or in the HEAD tag using the STYLE tag.
